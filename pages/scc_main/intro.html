Santa Clara County Active Contracts

This data comes from the two 'Active Contracts' reports created by the Procurement department every (or almost every) month. If you have any questions about this data, see the source PDF files linked below, the README and Data Issues. To see contracts for which contract documents are avaiable, see the browsable list and click on the "Docs" column header to sort by number of supporting documents. To see more context on issues with the data, go to the recent Santa Clara County Civil Grand Jury report entitled "Garbage In, Garbage Out: Santa Clara County Public Contract Data".
